make changes to a file

 git add --plus what files you actually changed     # Stages our changes

 git commit _m "changed something"    (# Commits our changes)

 git fetch upstream

If changes on the branch we’re based on (master)

 git rebase upstream/master

If conflicts:

 git mergetool

 enter diff

 git rebase --continue
